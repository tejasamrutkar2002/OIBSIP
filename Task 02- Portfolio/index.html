HOW IT WORKS?

While UI handles the look of a digital product, UX handles how all of the UI elements work together to create a seamless user experience. A UX/UI designer combines these two areas by carrying out user research first, and then implementing the findings in the visual design in the form of mockups, wireframes, and prototypes.
